POST/template/deleteTemplateKey

Checks to see if the request has the required authorization, which should have the form "templateKey:x;githubKey:y". If it passes, it checks whether or not the user's github key has access to the repository associated with the template. If both of these are true, it deletes the template from the template key table as well as the table of write keys for that template. Terminates with 401 if authorization fails.
